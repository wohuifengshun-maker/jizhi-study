# 极致 · 智学私教（Jizhi Study）

> 面向留学生的 AI 课后自学系统 · Newcore / 极致教育

学生在课后通过向 AI 私教提问来巩固知识点，最终达到通过课程考试的水平。即使零基础，也能通过本系统自学通过课程。

## 产品定位

不是"套壳 ChatGPT"，而是一个**绑定机构课程资料 + 押题 know-how + 进度追踪**的垂直自学系统。核心壁垒在于：

1. **绑定课程资料的精准问答**（RAG）— 基于本校本课的课件、真题、考点作答
2. **私教式教学逻辑** — 苏格拉底式追问、检验理解、对标考试
3. **进度追踪与考点对齐** — 把"自学"变成"能过考试"

### v5 新增 / 修复

- **🎯 测试我** — 随时点，AI 就已学考点出题，答完即时判对错
- **测一下（单考点）** — 考点掌握地图里每个学过的考点旁可单独自测
- **📝 小测验** — 一次 4 道选择题，做完出"得分 + 薄弱分析"，错题自动进薄弱雷达
- **答题对错判定** — 学生回答检验题时，AI 判对错；答错自动标记薄弱（修复此前只认关键词的问题）
- **学习足迹修正** — 正确记录学生提问与归属考点（修复标题错位、涉及考点为 0）

### v3 新增功能

- **👣 学习足迹** — 自动归档每次师生对话，按天分组，随时回看学过/问过什么
- **🎯 薄弱雷达** — 自动捕捉卡顿/答错/反复提问的考点，汇总薄弱清单并给出针对性复习建议，一键让私教重讲
- **数据持久化** — 学习记录、进度、薄弱标记本地保存，关闭后再打开仍在；含"重置演示"

### v2 功能

- **零基础引导** — 三步 onboarding：课程/学校、基础水平、每日可投入时间
- **AI 智能学习计划** — 自动生成对齐作业 due 的 7 天每日任务，含建议时长
- **资料上传** — 上传课件自动解析出考点并同步到进度地图
- **AI 检索资料** — 联网匹配本校本课的官方主页、教材、真题、视频
- **时间轴** — 作业与考试倒计时，临近自动标红

## 技术方向（正式版）

沿用机构现有技术栈：前端网页 + Supabase（账号/进度/记录）+ Cloudflare R2（课件存储）+ Claude API（问答与教学）+ pgvector（RAG 检索）+ Netlify（部署）。
